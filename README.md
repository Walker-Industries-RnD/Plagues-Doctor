### Quickstart

1. Launch Plague Doctor
2. **Create a new Plague system** → enter directory & project name
3. Generates:

   * Core interfaces & utilities
   * Platform-specific implementations
   * Security infrastructure
   * Example services

### Add Workers

Workers are elevated, platform-specific services:

1. **Add a new Plague worker** → choose project & name it
2. Creates:

   * Linux (Unix socket) & Windows (Named pipe) workers
   * Self-verifying integrity checks

### Features

* **Integrity Verification:** Workers self-check executable & dependencies
* **Local-Only Communication:** Named pipes (Windows) / Unix sockets (Linux)
* **Automatic Privilege Separation:** Minimal required privileges
